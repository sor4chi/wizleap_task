## getting start
```sh
$ git clone git@github.com:wizleap/laravel-beginner.git
$ cd laravel-beginner
$ npm install
$ docker-compose up -d
$ docker exec -it laravel_beginner_php7 /bin/bash -c 'composer install'
$ docker exec -it laravel_beginner_php7 /bin/bash -c 'php artisan key:generate'
$ open http://localhost:8083
```
